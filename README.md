# Rocket Playground: Solar Kids

A browser game for young kids: fly a rocket in space, visit the solar system, and collect stars.

## Run

Open `index.html` in a browser.

## Controls

- Keyboard: `Arrow Keys` or `WASD`
- Mouse: move pointer on canvas and rocket follows
- Touch: drag on canvas

## Notes

- Includes Sun, Earth, Moon, and all major planets.
- Sound is generated in-browser (no external audio files required).
- Optional upgrade links for graphics/audio are listed in `ASSET_RESEARCH.md`.
