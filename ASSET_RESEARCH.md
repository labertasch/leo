# Asset Research (March 8, 2026)

This project currently uses hand-drawn canvas graphics and procedural sounds, so it runs with no downloads.

If you want upgraded graphics/audio while keeping licensing simple, these are good sources:

## Graphics

- NASA media usage guidelines (planet photos/textures):
  - https://www.nasa.gov/nasa-brand-center/images-and-media/
  - NASA states media is generally usable for educational/informational use, with restrictions around logos, endorsement, and some third-party content.
- Kenney "Simple Space" (2D space assets, CC0):
  - https://kenney.nl/assets/simple-space
  - Kenney asset pages list `License: Creative Commons CC0`.
- SVG Repo licensing page (many open-license vectors/icons):
  - https://www.svgrepo.com/page/licensing/
  - Check per-icon license on each asset page before final use.

## Audio

- Pixabay content license summary (music + SFX):
  - https://pixabay.com/service/license-summary/
  - Free to use and modify, attribution not required; review prohibited uses.
- Freesound FAQ (license model overview):
  - https://freesound.org/help/faq/
  - Sounds can be CC0, CC-BY, or CC-BY-NC; confirm each sound's exact license.
- Downloaded into this project from OpenGameArt (CC0):
  - Rocket engine: https://opengameart.org/content/rocket-noise
  - Space rumble/atmosphere: https://opengameart.org/content/space-sounds

## Style Direction

To keep a "Futurama-like" mood without copying IP directly:

- Use chunky rounded shapes and saturated orange/teal/sky-blue contrast.
- Keep outlines simple and playful.
- Prefer original ship silhouettes and UI details over direct references to show assets.
